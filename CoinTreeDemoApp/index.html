CoinTree API demo app

Basic application that reads data from the public CoinTree price API
CoinTree API accessable from cointree.com.au
Displayed below is the current ask (buy) and bid (sell) price
The previous price is also shown with an indication of price rise and fall.
Price is updated automatically every 30 seconds
